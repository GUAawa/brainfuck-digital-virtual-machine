# brainfuck-digital-virtual-machine
使用logisim制作了数电的brainfuck

## 使用
使用py烧录器编写纯正的bf代码并存储为logisim的rom文件格式（可直接于logisim读取）

当然你乐意的话也可以直接写在rom里啦~

C++文件也是我做的模拟器，在拧麻花之前我先拿它试验了。 它 的 算 法 并 不 高 效 ！

## 废话
过程受到但不完全受到了《But how do it know?》的影响。

这个项目尝试了两次，第一次由于规划不充分，过程相当坎坷，电路更是拧成了麻花。因为设计受到了scottCPU影响，我甚至为控制单元设计了8个微时刻......
[]的逻辑设计有误，而麻花极大地影响了排查。最终由于无法找出问题部分而失败。

第二次提前进行了cpp模拟，并一口气连好了所有电路。虽然出现了几次问题，但是由于此次接线比较优雅~，且算法没有问题，很轻松的排查掉了。
